# Food - Make Tofu from Soybeans, Water, and Epsom Salt (large batch)

## Description
 Boil up Soybeans, grind into soymilk, filter off okra, and add Epsom Salt as Coagulants.&#10;&#9;&#9;&#9;gently stir and heat, the curds will separate.
